Project on Console

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.

Steps to setup Visual Studio Code-

1. Download and install VS Code.

2. Create a new file.

3. Install support for your favorite programming language.

4. Change your keyboard shortcuts and easily migrate from other editors using keybinding extensions.

5. Customize your editor with themes.

6. Explore VS Code features in the Interactive Editor Playground.
